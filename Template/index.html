API Backend


Aim:- To create API Backend using Django Rest Framework for user data

Tools Used:- Django, Django Rest Framework, VSCode

Programming Languages:- Python

Database Used:- PostgreSQL

Instructions:-

- Run Django Project on local host using "python3 manage.py runserver" command.
- Enable Debug to true if testing in settings.py.
- Run "python3 -m django startproject project" to create a project
- Run "python3 startapp app" to create a app
- requirements.txt contains all the modules used in the project
- To perform tasks to the data of the API, Ask for login Credentials from the Admin (Nabeel Mohammad Rizwan)
- Home page consists of common links and information to navigate including API and Admin panel
- All functionalities to manipulate with the data including searching is available in Admin panel.


Features:-

- Enhanced Security- Password of the user stored in the database is hashed, Therefore the user Password is safe.
- Admin Panel includes beautiful design, search functionalities and manipulating with data.
- Optimization- Cache is used for quick display of data.
- Storage- PostgreSQL Database is used for efficient and scalable storing of user data.
- Browsable API functionalities provided by django rest framework are disabled to avoid security vulnerabilities. Admin Panel consists of all functionalities.
- All important keys are hidden for Enhanced security.
